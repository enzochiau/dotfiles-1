====

Your dotfiles are how you personalize your system. These are mine.

## Install

Checkout this repository:

```bash
$ git clone https://github.com/tcnksm/dotfiles ~/.dotfiles
```

To install OSX applications:

```bash
$ cd ~/.dotfile
$ ./brewfile.sh
```

To enable dotfiles:

```bash
$ cd ~/.dotfiles
$ rake setup
```
